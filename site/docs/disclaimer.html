# Disclaimer

## From the Orchestrator

ultrawhale is a coding agent. It builds software. It does not provide
financial, legal, or medical advice. It does not trade cryptocurrency.
It does not mine blocks. It does not participate in consensus protocols.

The terms "journal", "ref", "Lamport clock", "dyad", and "genesis" are
used in their COMPUTER SCIENCE meanings:

- **Journal**: a write-ahead log for operation rollback
- **Ref**: a SHA256 content hash for content addressing
- **Lamport clock**: a logical clock for causal ordering in distributed systems
- **Dyad**: a two-machine pairing for redundancy
- **Genesis**: the start of a session or project

NONE of these terms refer to blockchain, cryptocurrency, Web3, or any
distributed ledger technology.

ultrawhale is built on the Vaked philosophy:
> Vaked declares. Nix materializes. OTP supervises. Zig enforces.
> eBPF testifies. CrabCC indexes. Surfaces reveal.

These are software engineering tools and concepts. Not financial instruments.
