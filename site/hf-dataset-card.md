# ultrawhale-dogfood

<p align="center">
  <img src="https://raw.githubusercontent.com/peterlodri-sec/ultrawhale/main/site/logo.svg" width="80" alt="ultrawhale logo">

**Self-generated dataset from the [ultrawhale](https://github.com/peterlodri-sec/ultrawhale) agentic coding system and the [vaked](https://vaked.dev) open-source AI ecosystem.**

Two live data streams, fully open, auto-updating every 6 hours.

---

## Streams

### `dogfeed/` — ultrawhale M3 loop output
Agent interactions generated by ultrawhale running on a 10-second loop using free OpenRouter models. Each record is a raw agent session: prompt, reasoning trace, code output, tool calls.

Files:
- `dogfeed-v1-initial.jsonl` — genesis sessions
- `dogfeed-v2-science.jsonl` — science/research mode
- `dogfeed-v3-enriched.jsonl` — enriched with block metadata

### `telemetry/` — vaked ecosystem usage signals
Anonymous structural metadata from the vaked public sites. No audio content. No PII. Each record is a session event batch from one of the source sites.

**Event types:** `page_view`, `mode_change`, `audio_source`, `substance_selected`, `trip_started`, `trip_ended`, `drop_triggered`, `wallpaper_toggled`, `autopilot_toggled`, `science_opened`, `scene_saved`, `scene_recalled`, `session_end`

Sources:
| Site | Description |
|---|---|
| [chat.vaked.dev](https://chat.vaked.dev) | G0DM0D3 — free-model AI chat proxy |
| [music.vaked.dev](https://music.vaked.dev) | ENTHEA — psychedelic visual synthesizer |
| [beat.vaked.dev](https://beat.vaked.dev) | Vaked-FM swarm avatar |
| [protocol.vaked.dev](https://protocol.vaked.dev) | AG-UI protocol + genesis spec |
| [irc.vaked.dev](https://irc.vaked.dev) | public IRC community (Ergo + Gamja) |

### `telemetry-consolidated.jsonl`
All telemetry batches merged into a single file. Updated every 6 hours by CI.

---

## MANIFEST-live.json
Machine-readable index updated on every CI run. Contains live record counts, file lists, source URLs, and genesis seal.
