# I Found This Thing Called ultrawhale and You Can `ls` the Agents??

ok so i was browsing github and found this project called ultrawhale. it's a coding agent, 52 blocks, 7 plugins, whatever. but then i saw this:

```
/vfs tree
📁 ultrawhale
├── 📁 orchestrator
│   ├── 📄 status → orch: did:key:orch:z5... | 12 turns | 3 agents
│   └── 📄 did → did:key:orch:z5bf8c...
├── 📁 agents
│   ├── 📁 swe-1
│   │   ├── 📄 role → swe
│   │   ├── 📄 status → running
│   │   └── 📄 caps → FULL
│   └── 📁 explore-2
│       └── 📄 caps → OBSERVE
├── 📁 dyad
│   └── 📄 peer → dev-cx53
└── 📄 SACRED → sacred: ● 1:1 · lag: 0ms
```

YOU CAN `ls` THE AGENTS. YOU CAN `cat` THEIR STATUS. THE CAPABILITY GRAPH IS A FILESYSTEM.

you can even `echo "fix the auth bug" > /ultrawhale/orchestrator/delegate` and it ACTUALLY DELEGATES THE TASK.

this is either the coolest or the most unhinged thing i've ever seen in a terminal. probably both. the readme says something about "Vaked philosophy" and "Context × Time × Space" and i don't fully understand it but i respect it.

52 blocks. 80 releases. built in one session. the agent built the agent.

anyway here's the repo: github.com/peterlodri-sec/ultrawhale

i'm going to go `cat /ultrawhale/brain/dump` now. wish me luck.
