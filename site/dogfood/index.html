🐕 ultrawhale DogFood Dataset

Public dataset of human↔LLM interactions. Collected in the background by the Dog Feed loop. Designed for LLM fine-tuning and alignment research.

—
samples collected (live)

Format

{"user_message":"...","free_response":"...","free_model":"google/gemma-3-4b-it:free","deepseek_response":"...","timestamp":"...","session_id":"v52.0.0"}

Usage

import json
with open("dogfeed.jsonl") as f:
    samples = [json.loads(line) for line in f]
# Fine-tune on user→response pairs

Download

Dataset files are available at ~/.ultrawhale/dogfeed/dogfeed-*.jsonl

Public mirror coming soon on Hugging Face Datasets.
