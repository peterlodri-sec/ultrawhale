## vaked-base

**vaked-base** is the foundation monorepo for the Vaked ecosystem. It contains:
- **vaked/** — the language (grammar, schema, examples)
- **vakedc/** — Python prototype compiler (parse → check → lower)
- **vakedz/** — Zig production compiler (cache-native)
- **docs/** — language design series, protocol RFCs, daemon specs
- **hosts/** — vakedos NixOS deployment configuration

ultrawhale lives inside vaked-base as a module (`docs/ultrawhale-README.md`).

## vaked

**vaked** is the capability-graph philosophy. It's the idea that software
should be DECLARED, not programmed. Seven layers:

```
Declares → Materializes → Supervises → Enforces → Testifies → Indexes → Reveals
```

vaked is not a product. It's not a company. It's a way of thinking about
software — as a graph of capabilities, not a sequence of instructions.

ultrawhale IS vaked made executable. Every block is a declaration.
Every engine is a layer. Every recursion is a proof.
